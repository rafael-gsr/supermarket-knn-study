# Introduction

The idea of this project is to simulate the supermarket queue classification.
Nowadays is usual that the supermarket have an kilometric line of customers waiting to pay.

The idea of this project is to say the queue should be the quick, high priority or standard.

## Types of queues
- quick queue 
- high priority
- standard

## Input data
Array with the following properties
````
` [ age, number of items, have good relationship ] ` 

### Description of the fields
- age: 70+ should have hight priority
- number of items: until 15 should go to quick
- have good relationship: scale from 0 to 5. 4+ clients should have high priority
